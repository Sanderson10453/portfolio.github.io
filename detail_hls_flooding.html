Flood Segmentation Model with Harmonized Land Satellite (HLS)

Using HLS imagery in a flood segmentation model to understand areas of Jacksonville prone to flooding.

Leveraging satellite imagery to highlight areas of Jacksonville prone to rain induced flooding

According to the research by FSU, Jacksonville should expect a 22% increase in extreme precipitation events under a 3.6 degree F climate warming scenario. Using Harmonized Land Satellite (HLS) and flood segmentation modeling, we can predict areas that flood and areas that stay dry during rain-induced flood. The Harmonized Land Satellite is a project from NASA that aims to produce a seamless surface reflectance record. Users can access these satellite images from NASA.

Process

To begin with, HLS images were downloaded the terminal. The HLS download can be run as a shell command with arguments that can specify the amount of cloud cover, date ranges, and spatial tiles to download. This can be see in the first photo; meanwhile, the second photo shows what a downloaded HLS image looks like.

Once downloaded, a transformer model from NASA-IBM was adaptated for flood segmentation in Jacksonville — some code snippets are show in the third photo. After feeding these HLS images into the model, we can see how the model views the various bands in the fourth photo.

Once run, the model will output an image that highlights areas where it "sees" flood waters. To highlight areas that we believe are prone to flash flooding, we compare flash flood days to dry days in Jacksonville. We can then compare areas that tend to be flooded on flash flood days, yet are labeled as dry land by the model on non-flood days.
